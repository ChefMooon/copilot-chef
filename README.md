# copilot-chef

A project to help you cook up amazing solutions with GitHub Copilot.

## Overview

This project demonstrates best practices and utilities for working with GitHub Copilot.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Usage

```bash
npm start
```

## Development

Run tests:
```bash
npm test
```

Run linter:
```bash
npm run lint
```
