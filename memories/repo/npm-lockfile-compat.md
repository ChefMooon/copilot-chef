- CI uses Node 20 (npm 10); lockfiles generated by npm 11 can cause `npm ci` EUSAGE with missing `@emnapi/core`, `@emnapi/runtime`, and `esbuild`.
- Fix by regenerating lock with npm 10: `npx -y npm@10.9.3 install --package-lock-only --ignore-scripts`.
- Verify with: `npx -y npm@10.9.3 ci --ignore-scripts`.
